# API Back-End para Gerenciamento de Pedidos – Raízes do Nordeste

Projeto desenvolvido para a disciplina de Projeto Multidisciplinar – Trilha Back-End, do curso de Análise e Desenvolvimento de Sistemas.

## Descrição

Esta API REST foi desenvolvida para simular o funcionamento de um sistema de pedidos da rede Raízes do Nordeste.  
O sistema permite cadastro de usuários, autenticação, cadastro e consulta de produtos, criação de pedidos, simulação de pagamento e atualização de status.

## Tecnologias utilizadas

- Node.js
- Express.js
- Prisma ORM
- SQLite
- JWT
- bcryptjs
- Git e GitHub
- Thunder Client

## Requisitos para executar

- Node.js instalado
- npm instalado
- Git instalado
